# VTOBForge Engineering Playbook

The Engineering Playbook documents the engineering principles, standards, architecture decisions, workflows, and technical practices used across all VTOBForge software projects.

Its purpose is to ensure that engineering knowledge becomes institutional knowledge rather than remaining dependent on individual memory.

## Status

🚧 Active Development

## Purpose

- Document engineering decisions.
- Standardize development practices.
- Improve consistency across projects.
- Preserve technical knowledge.
- Support builder education.

## Structure

- Development Environment
- Git Standards
- Flutter Standards
- Project Structure
- Architecture Decisions
- Deployment
- Engineering Retrospectives

## Philosophy

Build with purpose.

Document with clarity.

Improve continuously.
